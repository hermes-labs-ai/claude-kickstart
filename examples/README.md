# Scenario fixtures

These two fixtures are contract examples, not canned runtime menus. They demonstrate the specificity and diversity expected from the same possibility method.

- `mental-health-professional.json` centers safe self-reflection, language patterns, boundaries, and client education.
- `parent-household.json` centers fridge-first cooking, flexible household rhythms, family recipes, and family memory.

The automated test checks required domain anchors, forbidden-domain leakage, structural fields, mode diversity, and distinct labels. It does not claim that model wording is deterministic. The real Claude Code smoke test is reported separately in `TEST-EVIDENCE.md`.
