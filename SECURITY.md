# Security

Claude Kickstart is deliberately project-local. Please report suspected security problems privately through [GitHub's security advisory form](https://github.com/hermes-labs-ai/claude-kickstart/security/advisories/new) instead of opening a public issue.

Do not include credentials, private portraits, onboarding notes, or user creations in a report. Include only the minimum reproducible technical details.

The project does not use permission bypasses or modify global Claude Code settings. Claude Code's permission system and the operating system remain the final security boundaries.
